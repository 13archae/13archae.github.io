Moving Eyes Project
MIT MERN/Javascript Bootcamp

A set of eyes follows your mouse-pointer around the screen.
The eyes are generated purely using CSS. JavaScript "listens" for input from the mouse-pointer as it moves around the screen, and updates the location of the eye pupils accordingly.

Pac-Men Project
MIT MERN/Javascript Bootcamp

Generate and launch pac-men to bounce around your browser.
The pac-men stay within the bounds of the browser window, which is accomplished through a basic colision-detection system, which uses the inner width and height of the browser window to define "walls" to bounce the pac-men around. JavaScript is used to move the pac-men by continually updating their coordinates.

Boston Bus Tracker Project
MIT MERN/Javascript Bootcamp

Track busses on a Google Map in real-time along the bus-route that connects Harvard University's campus with the Massachusetts Institute of Technology campus in Boston.
This project uses Google Map API, as well as APIs hosted my the Boston Transit Authority to give real-time data on the bus-line tracked here.
