How to host your Code-folio site on GitHub... for Free!

  1.   Create a free account on GitHub
    1.   Navigate to https://github.com/join to create your account:
    2. You will need to:
      •     Create a User Name
      •     Provide a valid email address
      •     Create a password
      •     Solve a puzzle to prove your humanity
      •     Click the "Create" button

        This action launches a modal window where you will enter the MFA code sent to your email

    3. You will need to:
      •     Open your email client
      •     Navigate to the email account listed on the screen (e.g. the email with which you created your account)
      •     Find the email from GitHub containing the Launch Code
      •     Enter the Launch Code in the fields provided

        If you enter the correct launch code, the process of creating your GitHub accout will finalize.

        This process may take a moment or two, but when finished, you will be on the GitHub Dashboard for your account.

  2. You will need to:
    •     Find the "Start a new repository" section of the Github Dashboard
    •     Enter a file name in the format:   <username>.github.com
    •     Make sure the repository is marked "public" so your page will be hosted by Github Pages
    •     Click the "Create New Repository"

      This action will create your new repository

      Once created, you will be navigated to the page for your new repository

    1. You will need to:
      •     Find the Create a New File link as shown above on the repository page
      •     Click the link

        This action will result in the creation of a new file

        You will be navigated to the Edit page for your new file

    2. You will need to:
      •     Name the file index.html
      •   Type <h1>Hello, Chris!</h1> into the editor
      •     Click the "Commit Changes" to to initiate the save of your code to the repository

        This will launch a modal window to enter your Commit Comment

    3. You will need to:
      •     Enter a meaningful comment about the file(s) e.g Something that will identify the commit in human terms
      •     Click the "Commit Changes" to to complete the save of your code to the repository

        Finishing the commit will return you to the repository page

  4. You will need to:
    •     You can see that the file now exists in the repository
    •     The comment you entered for the commit is also visible

      Next, you will view the Actions tab on the repository page to check the status of the deployment

  5. You will need to:
    •     The workflow above is complete.
    •     The green dot icon shows the workflow-run was completed successfully

      Now we can check-out our webpage in the browser

  6. You will need to:
    •     We are now able to view the rendered site in the browser!

      If you enter the correct launch code, the process of creating your GitHub accout will finalize.

     This process may take a moment or two, but when finished, you will be on the GitHub Dashboard for your account.

Wrapping-up:
  •     Now, you can repeat Steps 1-6 to add more content to your repository, and thus your Code-folio.
  •     To learn more about GitHub Pages, navigate to GitHub Pages Documentation
  •       The doccumentation is also an excellent place to learn more about GitHub and all of the amazing tools the platform offers

   I am hoping you found this page useful.

   This GitHub service, free for all, eliminates the need to buy a domain, pay for hosting, etc. when you are just starting-out.

   GitHub is truly a platform built for developers!
